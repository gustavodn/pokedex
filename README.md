# Instructions

1. Clone the repository
2. run 
```
npm install 
```
3. install Nest CLI
```
npm i -g @nest/cli
```
4. Up database
```
docker-compose up -d
```
5. Clone the file __.env.example__ and rename the __.env__ file.

6. Fill out the missing environment variables

7. Run the app in ```Dev```:
  ```
  npm run start:dev
  ```

8. Run seed
```
http://localhost:3000/api/seed
```
